TimeTracker Pro Setup

Get your time tracking system ready in minutes

Step 1: Firebase Configuration

1. Go to Firebase Console

2. Create a new project or select existing one

3. Enable Firestore Database

4. Copy your configuration and replace the placeholder in index.html:

const firebaseConfig = {
  apiKey: "your-api-key",
  authDomain: "your-project.firebaseapp.com",
  projectId: "your-project-id",
  storageBucket: "your-project.appspot.com",
  messagingSenderId: "123456789",
  appId: "your-app-id"
};

Step 2: Firestore Security Rules

Set up basic security rules in Firestore (found in firestore.rules file):

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /timeRecords/{document} {
      allow read, write: if true;
    }
  }
}

Note: For production, implement proper authentication and stricter rules.

Step 3: Customize Staff Names

Edit the staff dropdown in index.html to match your team:

<option value="Your Name">Your Name</option>

Step 4: Local Development

For local testing, use a simple HTTP server:

# Using Python
python -m http.server 8000

# Using Node.js
npx http-server

# Using PHP
php -S localhost:8000

Step 5: Deploy (Optional)

Deploy to Firebase Hosting:

npm install -g firebase-tools
firebase login
firebase init hosting
firebase deploy
Launch TimeTracker Pro

Pro Tips:

  • The app works offline and will sync when connection is restored
  • Location tracking requires HTTPS in production
  • Consider adding user authentication for production use
  • Monitor your Firebase usage to avoid unexpected costs
